# LinguaAI Privacy Policy Repository

This repository publishes the public privacy policy page used in Google Play Console:

- Live URL: https://aabbcdl.github.io/linguaai-app-privacy/
- Source of truth: `index.html`

## Language Switch

The page includes a language selector (English / 简体中文 / Bahasa Indonesia / ไทย / Tiếng Việt / Bahasa Melayu), implemented in `index.html` and remembered with `localStorage`.

## Update Flow

1. Edit `index.html`
2. Commit and push to `main`
3. Wait for GitHub Pages rebuild
4. Verify the live URL
