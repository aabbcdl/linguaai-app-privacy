# LinguaAI Privacy Policy Repository

This repository publishes the public privacy policy page used in Google Play Console:

- Live URL: https://aabbcdl.github.io/linguaai-app-privacy/
- Source page: `index.md`

## Language Switch

The page includes a language selector (English / 简体中文) implemented in `index.md` using lightweight inline HTML + JavaScript and stores user preference in `localStorage`.

## Update Flow

1. Edit `index.md`
2. Commit and push to `main`
3. Wait for GitHub Pages to rebuild
4. Verify the live URL
